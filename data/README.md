# Data Directory

Folder ini menampung semua artefak data mentah yang dibutuhkan oleh proyek.

- `n8n/database/` — skrip SQL dan utilitas backup untuk Postgres (schema, sample queries, dsb) yang digunakan oleh workflow n8n.
- `n8n/test-payload.json` — payload contoh untuk menguji aliran n8n dari aplikasi web.

Gunakan path relatif dari root repository (misalnya `data/n8n/database/schema.sql`) ketika menjalankan perintah dari dokumentasi.
